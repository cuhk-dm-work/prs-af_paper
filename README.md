1) Filename: plink_score_commands.sh
- Includes PLINK v2.0 command for computing the PRS-AF scores.

2) Filename: test_ph_assumption.R
- Includes R code for the Schoenfeld residuals test to assess the proportional hazards assumption.

3) Filename: competing_risk_analysis.R
- Includes R code for competing risks of death analysis using Fine-Gray models.

4) Filename: compute_c-index_nri_idi.R
- Includes R code for computing the concordance index (C-index), category-free continuous net reclassification improvement (NRI) index, and integrated discrimination improvement (IDI) index.
- Calculates the 95% confidence intervals for all indices using 10,000 iterations of bootstrap resampling or perturbation-resampling.

5) Filename: compute_brier_score_calibration.R
- Includes R code for computing the Brier score to evaluate whether predicted risks correspond to observed event rates (calibration assessment)
